## 🛠️ 使用指南
1️⃣ **精准搜索**  
▸ 使用完整游戏名/英文原名效果最佳  
▸ 示例：`千恋万花` → `Senren＊Banka`  

2️⃣ **结果筛选**  
▸ 优先选择<span style="color:#4CAF50">绿色标签</span>平台（免登录+直链下载）  
▸ 金色平台需配置代理，白色平台需完成对应条件  

3️⃣ **下载须知**  
▸ 推荐使用IDM/FDM等下载工具加速  
▸ 遇到Cloudflare验证时耐心等待5秒
